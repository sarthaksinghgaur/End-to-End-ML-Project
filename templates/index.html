Diabetes Prediction Web App

Powered by Machine Learning and Flask

About This Project

This web application predicts whether a person is likely to be diabetic based on their health information. The model is trained on real-world medical data and leverages machine learning classifiers for accurate results.

  • Tech Stack: Python, Flask, HTML/CSS, scikit-learn
  • Features: Input validation, model pipeline, user-friendly interface
  • Use Case: Educational, Screening Aid, Demonstration of Data Science skills
